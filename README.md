# [Blender](https://github.com/fragement-contrib/Blender)
使用免费开源好用的[Blender](https://www.blender.org/)制作的一些模型等3D相关内容

## 如何使用？

首先你需要安装依赖：

```
npm install
```

然后启动开发服务器：

```
npm run dev
```

浏览器打开： http://127.0.0.1:20000/#/

开发完后需要发布，直接：

```
npm run build
```

打包后的完整项目就是docs，直接打开docs/index.html即可。
